# Depression Analysis Dashboard

A data analytics and visualization project that generates synthetic student wellness data and analyzes depression risk factors using Streamlit.

## Open in Google Colab

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/0xSah/Depression-Analysis-Dashboard/blob/main/Depression_Analysis_Prototype.ipynb)

## Features

- Synthetic student dataset generation
- Depression risk scoring
- Correlation analysis
- Interactive Streamlit dashboard
- K-Means clustering
- Behavioral and academic trend visualization

## Technologies

- Python
- Pandas
- NumPy
- Streamlit
- Plotly
- Matplotlib
- Seaborn
- Scikit-learn
